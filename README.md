# To-Do List in C

A simple command-line To-Do List application built using the C programming language.

## Features

* Add new tasks
* View all tasks
* Delete tasks
* Mark tasks as completed
* Simple menu-driven interface

## Concepts Used

* Arrays and Strings
* Functions
* Loops
* Conditional Statements
* User Input Handling

## How to Run

Compile:
gcc todo.c -o todo

Run:
./todo

## Example Usage

Menu:

1. Add Task
2. View Tasks
3. Delete Task
4. Mark Task as Done
5. Exit

Example Tasks:

1. [ ] Study C
2. [ ] Complete Homework
3. [✔] Exercise

## Project Purpose

This project was created to practice core C programming concepts by building a simple task management application.
